Welcome to the Project Home page :)

So this is what you'll see on the home page.

On loading the home page, the browser will ask to share your location. Please select ALLOW on the browser to share your computer's location (else the map wont load). On allowing to share the location, the home page will be loaded fully, and the user's current location is indicated on map by blue marker.

The home page will give the user to select various options from the top bar, or a user can input a place name in the search textfield and hit the search button to search places.

On scrolling down, you'll be able to see reviews of places visited by various site users.  Initially, on loading you wont see any thing in the division for reviews about a place given by google users.

Essentially this is the only page of the project, loaded with all the functionality.

Due to this users don't have to navigate through various pages to search for another place using another feature of the project.

The project uses place search library of Google maps API to search for various places requested by the user.

By default, when the homepage is loaded for the first time, the textsearch feature is initiated.

Technologies used:

  • HTML5 geolocation to take the user's current location

  •  Google maps to display the location of the user on maps

  • LINQ to SQL to display the site user's comments
