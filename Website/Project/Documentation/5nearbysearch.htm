Nearby Search feature

The project allows the user to search for the places nearby within a radius. The user's current location is obtained by the browser and the places are search nearby the current location of the user.

If the user knows the name of the plaec he/she wants to visit for example Starbucks, Chipotle etc, he can use the nearby search utility to find that place nearby. The nearby search sorts the results in ascending order by their distance from the user's location. The search result is limited to the radius of 50km.

To search for the nearby places, the user must Check the checkbox saying "Search nearby" and type the name of the place in the search box or can select one of the option from top dropdown menu the which various types of places to search for.

How do we get the results:

The request given by user is sent to the Google's places library to search for the mathcing places with given parameters and the results are obtained in the JSON format. The useful data is filtered and displayed on the map. Initially first 20 results are displayed on the map as well as on the side panel.

To list obtain more places, user needs to click the button saying "More similar places" at the bottom of the list. On clicking, the next list of 20 places is displayed on the side panel replacing the initial results. The user can view upto 60 places.

 

Technologies used :

  • CSS3
  • javascript for taking user's input and requesting the data
  • JSON parsing for extracting the useful results
