Autocomplete feature

The project allows the user to search for the single particular place using the Autocomplete feature provided by the Google Places library. When the user opts to use the autocomplete feature by checking the "Autocomplete" checkbox, text entered in the searcbox is monitored for the changes and upto 5 matching results are suggested to user.

As text is entered, Autocomplete returns Place predictions in the form of a drop-down pick list. User can use the Places Autocomplete feature to help users find a specific location, or assist them with filling out address.

When a you select a Place from the list, information about the Place is returned to the is displayed on the maps as well as on the panel.

Note that when you start typing the place in the text box using the autocomplete feature, the places suggested are the places nearby your current location.
