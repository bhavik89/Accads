Project Synopsis

The project: Find Places is all about searching the places depending on your mood or your need!

This website is basically designed  for letting the user to search for any place in his/her vicinity. The website gives user various options to select to search for places. The current location of the user is obtained by the browser and based on the inputs given by user for search.

Options provided by the site to users:

  • Search for places by name
  • Search for places by any text input
  • Search for nearby places
  • Use the autocomplete to search for a particular place
  • Search by selecting one of the options depending on the need

Depending on the inputs given by the user, the search returns the list of places found for a search and plots it on the maps giving each place a marker on map.

Once the places are plotted on the map, the user can get the more information about a particular place by selecting it on the list of places or the place marker on the map. On selecting the place the site returns the reviews and photos of that place (if there are any). Also, it provides a link to get more information about the place.

This site allows the logged in users to post there reviews about the particular place, where the have been to and let others know about the places they have been to.

Technologies used for the project:

  • ASP.NET 4.0
  • C#
  • Javascript
  • JQuery
  • LINQ
  • HTML
  • CSS
  • AJAX
  • JSON
  • SQL
