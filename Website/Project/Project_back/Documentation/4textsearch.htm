The Textsearch feature

When the page is loaded for the first time, the text search feature is initialted. The project uses place search library of Google maps API to search for various places requested by the user.

The Google Places, Text Search Service returns information about a set of Places based on any text search, for example

  • "Restaurants nearby" or

  • "Museums in boston" or

  • simply any name of place like "Starbucks".

 The service responds with a list of Places matching the text string and any location bias that has been set. The search response returns the list of first 20 places matching the search string.

To list obtain more places, user needs to click the button saying "More similar places" at the bottom of the list. On clicking, the next list of 20 places is displayed on the side panel replacing the initial results. The user can view upto 60 places.

Alternatively, the user can select the type of place from the top dropdown menu. The top menu lists out various options (types of places) for the user to select.

How the results are obtained:

When the user inputs any text in the search box, a request with the query string and the user's current location is sent to the textsearch service of Google Places library and the results are obtained in the JSON format.

The JSON result is then parsed and the useful information about the place like name, address, Google users ratings etc is displayed on the side panel.
