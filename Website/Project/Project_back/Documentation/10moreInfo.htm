MoreInfo Page

The more info page displays the additional information about the place selected. This page displays the name of the place, addres, phone number, ratings, the price level of place, Google user comments about the place and photos of the same.

This page also displays the location of the place on the map.

To get more info. about the place, user must select a place on the map by clicking the marker on maps or by selecting a place on the side panel and click on "More info. on this place" next to "More Similar places" button.

How the details are fetched:

To get the more informaition about the place, the place's reference ID obtained by the search query is required. (Note that each place has a unique reference ID). Once we have a reference from a Place Search, we can request more details about a particular establishment or point of interest by initiating a Place Details request.
